# LED Slide Pro 📟

LED Slide Pro เป็นเว็บแอปพลิเคชันรูปแบบ Single-file HTML ที่ออกแบบมาเพื่อจำลองหน้าจอ LED ป้ายไฟวิ่ง (LED Matrix/Marquee Display) ที่มีความสมจริงสูง เหมาะสำหรับการเปิดใช้งานบนโทรศัพท์มือถือและ iPad เพื่อใช้เป็นป้ายไฟเชียร์ ป้ายประกาศ หรือป้ายต้อนรับแบบดิจิทัล

## 🌟 ฟีเจอร์หลัก (Features)

- **One-File Setup:** ใช้งานง่ายมาก โค้ดทั้งหมด (HTML, CSS, JS) รวมอยู่ในไฟล์ `index.html` ไฟล์เดียว ไม่ต้องพึ่งพาเซิร์ฟเวอร์หรือฐานข้อมูล
- **Realistic LED Effect:** ดีไซน์หน้าจอให้เหมือนหลอดไฟ LED ของจริง พร้อมเอฟเฟกต์แสงเรืองแสง (Glow) และเส้นสแกน (Scanlines)
- **Slide Management:** 
  - สามารถเพิ่ม (Add), แก้ไข (Edit) และลบ (Delete) ข้อความในแต่ละสไลด์ได้อย่างอิสระ
  - กำหนดเวลาแสดงผลแต่ละสไลด์ได้ (เป็นวินาที)
- **Customizable Effects:**
  - **นิ่ง (Static):** แสดงข้อความค้างไว้
  - **เลื่อน (Scroll):** ป้ายไฟวิ่ง สามารถเลือกระดับความเร็วในการวิ่งได้ถึง 5 ระดับ
  - **กระพริบ (Blink):** ข้อความกระพริบดึงดูดสายตา
  - เลือกสีตัวอักษรได้อย่างอิสระ (Color Presets)
- **Smart Orientation:**
  - **โหมดแนวนอนอัตโนมัติ:** เมื่อถือโทรศัพท์ในแนวตั้ง (Portrait) ตัวแอปสามารถถูกตั้งค่าให้หมุนหน้าจอ 90 องศาโดยอัตโนมัติ เพื่อให้ผู้ใช้ถือโทรศัพท์ได้ถนัดมือพร้อมกับโชว์ป้ายไฟในแนวยาว
  - รองรับการปรับโหมดเป็นแนวตั้ง/แนวนอนได้ตามต้องการ
- **Intuitive Controls:**
  - **Touch Swipe:** รองรับการใช้นิ้วปัดหน้าจอเพื่อเปลี่ยนสไลด์ (ปัดตามทิศทางหน้าจออย่างชาญฉลาด)
  - แผงควบคุมดีไซน์แบบ Glassmorphism สวยงามและไม่บังเนื้อหา
- **Data Persistence:** บันทึกข้อมูลสไลด์ทั้งหมดลงในตัวเครื่องอัตโนมัติผ่าน `localStorage` ปิดแอปแล้วเปิดใหม่ข้อมูลก็ไม่หาย
- **Fullscreen Mode:** รองรับการแสดงผลแบบเต็มหน้าจอเพื่อความสมจริงขั้นสุด

## 🚀 วิธีการใช้งาน (How to use)

1. ดาวน์โหลดหรือคัดลอกไฟล์ `index.html` ไปไว้ในเครื่อง
2. ดับเบิ้ลคลิกไฟล์ `index.html` เพื่อเปิดผ่านเว็บเบราว์เซอร์ (แนะนำ Google Chrome หรือ Safari)
3. กดปุ่มไอคอน **ฟันเฟือง (Settings)** ที่มุมขวาล่างเพื่อเปิดแผงควบคุม LED Manager
4. ทำการเพิ่มสไลด์, เลือกสี, เอฟเฟกต์ และปรับตั้งค่าต่างๆ ตามต้องการ
5. ปิดแผงควบคุม แล้วนำไปใช้งานได้เลย!

## 📱 การใช้งานบน Mobile / iPad

แอปนี้ถูกออกแบบมาเพื่อ Mobile-First โดยเฉพาะ:
- คุณสามารถนำไฟล์ `index.html` ไปเปิดใน Safari บน iPhone/iPad
- หากคุณเลือกทิศทางแบบ **แนวนอน (Horizontal)** ระบบจะหมุนข้อความให้อัตโนมัติเมื่อคุณถือเครื่องแนวตั้ง ทำให้คุณสามารถถือมือถือด้วยมือเดียวได้สะดวกเวลาโชว์ป้ายไฟ
- สามารถใช้นิ้วปัดหน้าจอ (Swipe) เพื่อเปลี่ยนไปสไลด์ถัดไปหรือย้อนกลับได้อย่างรวดเร็ว
